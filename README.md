# Unified Task Manager Pro

A powerful, single-file task management application that combines the strategic planning of the **Eisenhower Matrix** with the long-term organization of a **Monthly List**.

## Key Features

- **Dual Views**: Switch instantly between Matrix (Strategy) and List (Timeline) views.
- **Unified Data**: Tasks added in one view are accessible in the other.
- **Premium UI**: Glassmorphism aesthetic, smooth transitions, and distinct quadrant colors.
- **Smart Reminders**: Get browser notifications 15 minutes before any task deadline.
- **Recurring Tasks**: Easily duplicate tasks for next week or next month.
- **Archive**: Automatically groups past months' tasks at the bottom of the List view.

## How to Deploy

Since this is a single HTML file, deployment is instant.

### Option 1: Netlify Drop (Recommended)
1. Go to [Netlify Drop](https://app.netlify.com/drop).
2. Drag and drop this entire folder (`task-manager-v2`) onto the page.
3. You will get a live URL immediately.

### Option 2: GitHub Pages
1. Upload `index.html` and this `README.md` to a GitHub repository.
2. Enable GitHub Pages in the repository Settings > Pages.

## Usage
Simply open `index.html` in Chrome, Edge, Safari, or Firefox. No internet required for local use.
